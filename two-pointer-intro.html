Two-Pointer Starter Guide

Three quick pages: definition → patterns → examples

What is the Two-Pointer Technique?

Two pointers means keeping two indices (often left and right) and moving them based on a rule until a condition is met. It often turns quadratic scans into linear time.

Why it’s fast

  • Each pointer moves at most n times → O(n)
  • Great for sorted arrays/strings and pair/interval problems
  • Saves extra memory vs brute force

Common moves

  • Opposite Ends: move inward (pair sum, container, palindrome)
  • Same Direction: slow/fast (remove dups, partitioning)
  • Meet-in-the-middle: search for first/last mismatch

Core Patterns & When to Use

1) Opposite Ends (sorted or ordered data)

  • Rule: if sum too small → move left++; if too big → right--
  • Examples: Two Sum (sorted), Container With Most Water, Palindrome checks

2) Same Direction (slow/fast)

  • Rule: fast explores; slow lags to build answer
  • Examples: Remove Duplicates in Sorted Array, partitioning by condition

3) Bidirectional Matching

  • Compare characters/items from both ends; meet in the middle
  • Examples: Valid Palindrome (with/without one deletion)

When to reach for Two Pointers

  • Sorted array/string; need a pair/interval answer
  • Can decide the next move from local info (no backtracking)
  • Want O(n) or O(n log n) vs O(n²)

Worked Examples (Java)

Two Sum (sorted)

public static int[] twoSumSorted(int[] nums, int target) {
  int left = 0, right = nums.length - 1;
  while (left < right) {
    int sum = nums[left] + nums[right];
    if (sum == target) return new int[]{ left, right };
    if (sum < target) left++;
    else right--;
  }
  return new int[]{ -1, -1 };
}

Palindrome Check

public static boolean isPalindrome(String s) {
  String t = s.replaceAll("[^A-Za-z0-9]", "").toLowerCase();
  int left = 0, right = t.length() - 1;
  while (left < right) {
    if (t.charAt(left) != t.charAt(right)) return false;
    left++; right--;
  }
  return true;
}

Container With Most Water

public static int maxArea(int[] h) {
  int left = 0, right = h.length - 1, best = 0;
  while (left < right) {
    int area = Math.min(h[left], h[right]) * (right - left);
    if (area > best) best = area;
    if (h[left] < h[right]) left++;
    else right--;
  }
  return best;
}

Bonus: Remove Duplicates (in-place)

public static int removeDuplicates(int[] nums) {
  if (nums.length == 0) return 0;
  int slow = 0;
  for (int fast = 1; fast < nums.length; fast++) {
    if (nums[fast] != nums[slow]) {
      nums[++slow] = nums[fast];
    }
  }
  return slow + 1;
}
